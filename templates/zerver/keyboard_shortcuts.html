Keyboard shortcuts

Navigation
/ Initiate a search
q Search people
Up or k Previous message
Down or j Next message
PgUp, K Scroll up
PgDn, J, Spacebar Scroll down
End Last message
Composing messages
Enter or r Reply to message
R Reply to author
c New stream message
C New private message
Tab then Enter Send message
Esc Cancel compose
Narrowing
s Narrow by stream
S Narrow by topic
v Narrow to all private messages
A or D Cycle between stream narrows
Esc Return to home view
Menus
i Open message actions menu
i then Enter Edit a message you sent
? Show these keyboard shortcuts
